BRIEF GAME DESCRIPTION 

The game initialized with 5 files: GETCH,HERO,MAZE,GOBLIN,MONSTER  and  PLAYGAME.It contains 4 classes one in each file except  maze file has no class and several sub classess in each class.
Hero class contains  the hero character and the ACTION method where the possible scenarios happen when the hero moves or interacts with monsters , goblins , walls and empty grid.Hero movement happens with A,D,W,S.If 
Class Getch was given to us and no modification were made from that file. 
The maze file initialized the maze and prints it on the enviorment and console.
Goblin class contains the Goblin but none of the methods  except the return of the coordinates of the goblin interation are there.
Class Monster contains the monster but it doesnt contain the the methods of the interaction between the monster and the hero except the return of the coordinates .
